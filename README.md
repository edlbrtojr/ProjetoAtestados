# ProjetoAtestados
Projeto de site que cadastra atestados e justificativas de falta dos alunos em uma escola pública.

Os dados são coletados e enviados para uma planilha google, através da API da sheetmonkey, onde os atestados são tratados e separados utilizando filtros da própria ferramenta. 

Este é um trabalho em andamento. O próximo passo é fazer uma aplicação web com o mesmo propósito, trazendo mais funções para o programa.
